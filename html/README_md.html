    D E K Я Y P T O S¶ ↑

    About¶ ↑

    These are little bits o' Ruby magic I've been using to mess around with decrypting parts of the Kryptos sculpture.

    Dependencies¶ ↑

    • Ruby (2.0+).

    Installation¶ ↑

    $ gem install dekryptos
